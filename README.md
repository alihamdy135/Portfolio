## 🚀 Overview

This portfolio is built as a highly optimized, zero-build-step single HTML file. It utilizes Tailwind CSS via CDN for rapid, responsive styling and Vanilla JavaScript for interactive elements. 

The design emphasizes clean typography, smooth animations, and scannability to effectively present my technical stack, featured projects, and professional experience as a Full Stack & Embedded Developer.

## ✨ Features

* **Zero Build Steps:** The entire project runs from a single `index.html` file with embedded CSS and JavaScript.
* **Fully Responsive:** Pixel-perfect rendering across mobile, tablet, and desktop viewports using Tailwind container queries and flex/grid layouts.
* **Custom Animations:** * Animated typewriter effect in the hero section.
  * Intersection Observer API for scroll-spy navigation highlighting.
  * RequestAnimationFrame for smooth, dynamic stat counters.
  * Staggered scroll-reveal animations for section content.
* **Pure CSS/HTML UI Mockups:** Features a custom-built, CSS-only dashboard mockup in the featured project card (no images or canvas used).
* **Interactive Contact Form:** Built-in client-side validation and success state handling using Vanilla JS.

## 🛠️ Tech Stack

* **Structure:** Semantic HTML5
* **Styling:** [Tailwind CSS](https://tailwindcss.com/) (via CDN with arbitrary values and custom theme configuration)
* **Interactivity:** Vanilla DOM JavaScript (No external libraries like jQuery or GSAP)
* **Typography:** [Satoshi](https://www.fontshare.com/fonts/satoshi) (Headings), [Inter](https://fonts.google.com/specimen/Inter) (Body), and JetBrains Mono (Code/Footer)
* **Icons:** [Material Symbols](https://fonts.google.com/icons) & [Devicons](https://devicon.dev/)
